标题

基于XXX模型的XXX问题研究

摘要

不要加图或者表或者公式，不超过一页800-1000字
![[Pasted image 20230918185847.png]]
开头模板：本文主要研究了XXX问题，根据XXX,利用XXX模型/方法，求解出XXX
摘要开头段不要写详细求解结果
![[Pasted image 20230918190325.png]]
中间段模板：针对问题一，根据/考虑XXX….,建立XXX模型/利用XXX方法，求解出XXX
优化类，预测类和物理类要明确数值，若数据过多可说明数据见附录
![[Pasted image 20230918190517.png]]
总结：
若写完后超过一页，可以不写
不要累赘重复前面的内容
加一些自夸

正文要求

不要超过三级标题
表格要用三线表
图和表都要有标题，表的标题在表的上方，图的标题在图的下方
重要公式右侧要有编号

公式

公式插入可以用mathpix
右侧编号的可以见建一个1x2的表格，左边填公式，右边插入序号
( 插入->文档部件 -> 域 ->autonum -> (1) )
后边再编写公式时直接复制即可

表格

表的标题在表的上方（带序号）自动编号

插入表格
将样式应用于整个表格
先选择无边框，再选择上框线和下框线
格式中设置表格属性居中
将样式应用于标题行，选择下框线

插入表的标题
（引入 ->插入题注 ->域 ->(1) )

分页符

该符号后面的内容会到下一页中
（插入 -> 分页）

参考文献

格式规范
去除首行缩进，按正文出现顺序编号
加角标（WPS->开始+角标）
